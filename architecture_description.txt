Ingestion: Python scripts pulling from Binance (REST), Reddit (PRAW), and NewsAPI.


Storage: Real-time sync to a Dockerized PostgreSQL instance.


NLP Brain: Mistral 7B (Ollama) performing few-shot sentiment classification.

Synthesis: An "Explainable AI" layer that triggers desktop alerts only when Sentiment and Price trends align.
